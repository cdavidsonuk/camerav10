# GitHub Pages Deployment

1. Extract the ZIP.
2. Open the extracted project folder.
3. Upload its contents to the repository root.
4. Confirm `index.html` is in the repository root.
5. Commit the files.
6. In GitHub, open **Settings → Pages**.
7. Deploy from the main branch and root folder.

Three.js is loaded from jsDelivr, so an internet connection is required.
