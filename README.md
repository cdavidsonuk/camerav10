# PhotographyCourses Pro 2.0 — Realistic Camera Rebuild

This release replaces the old CSS camera mock-up with a new original DSLR-style body built as a detailed interactive SVG.

## What changed

- Completely new original camera body
- Correctly integrated rear controls
- Detailed grip texture, seams, bevels and body contours
- Visible screws and hardware
- Hot shoe and pentaprism housing
- Realistic viewfinder eyecup
- Integrated top LCD
- Integrated mode dial and power switch
- Rear command wheel and SET button
- AF-ON, MENU, INFO and Q controls
- Live scene displayed inside the camera LCD
- Existing exposure engine, challenges, capture review and side controls retained
- Working shutter, power switch and command wheel

The camera is deliberately original and does not copy a specific manufacturer.
