# PhotographyCourses Pro 2.0 — Realistic Camera Body V2

This package contains a new high-detail original vector camera body designed specifically for the online virtual camera.

## Improvements

- Standalone reusable camera asset: `assets/camera-body-realistic.svg`
- More dimensional body shape and bevelled chassis
- Textured rubber grip and fine body texture
- Detailed hot shoe, viewfinder housing and top LCD
- Integrated screws, seams, buttons, dials and command wheel
- Controls positioned directly over their matching body parts
- Live-view LCD positioned inside the physical screen bezel
- Working mode dial, power, MENU, INFO, AF-ON, Q, rear dial and shutter
- Challenge panel now has a close icon and a reopen button

The body is an original design and does not copy a particular camera brand.
